# Polymarket Price Tracker

## 📌 Описание проекта

Сервис на **Java + Spring Boot**, который получает данные о рынках с Polymarket через HTTP API, отслеживает изменения цен и сохраняет их в базу данных.

Основная задача — реализовать устойчивую и расширяемую архитектуру обработки потоковых данных с внешнего API.

---

## ⚙️ Функциональность

* Получение данных о рынках с Polymarket (Polling через HTTP API)
* Отслеживание изменений цены для нескольких рынков
* Кэширование последнего значения цены 
* Сохранение изменений в PostgreSQL

---

## 🧱 Архитектура

```
Polling (Scheduler)
        ↓
HTTP Client (Polymarket API)
        ↓
PriceService
   ↓           ↓
Cache         Database
```
